# <strong>ゲームジャンル別売り上げランキングサイト</strong>
ゲームの売り上げをランキング形式で表示させる検索機能を備えたwebサイトを作りました。
制作期間は約三ヶ月半です。

---
## 開発環境
* Docker

### 実行方法
~~~sh
$ docker-compose up -d
~~~

### アクセス画面
APP_URL: http://localhost:8080
